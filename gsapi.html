gsapi package¶

The GS_API is a Python/C++ library for manipulating musical symbolic data. all modules are documented, for more infos on the given module type :

>> help(gsapi.modulename)

other useful resources :

  • online tutorials and documentation : https://giantsteps.github.io/GS_API
  • source code : https://github.com/GiantSteps/GS_API
