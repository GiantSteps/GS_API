gsapi.GSIO module¶

gsapi.GSIO.fromMidi(midiPath, NoteToTagsMap, TagsFromTrackNameEvents=False)[source]¶

loads a midi file as a pattern

Parameters:
  • midiPath – midi filePath
  • NoteToTagsMap – dictionary converting pitches to tags noteMapping maps classes to a list of possible Mappings, a mapping can be: a tuple of (note, channel) if one of those doesnt matter it canbe replaced by ‘*’ character an integer if only pitch matters for simplicity one can pass only one integer (i.e not a list) for one to one mappings if midi track contain the name of one element of mapping, it’ll be choosed without anyother consideration
  • TagsFromTrackNameEvents – use only track names to resolve mapping, useful for midi contained named tracks
gsapi.GSIO.fromMidiCollection(midiGlobPath, NoteToTagsMap, TagsFromTrackNameEvents=False, desiredLength=0)[source]¶

loads a midi collection

Parameters:
  • midiGlobPath – midi filePath in glob naming convention (e.g ‘/folder/To/Crawl/*.mid’)
  • NoteToTagsMap – dictionary converting pitches to tags (see gsapi.GSIO.fromMidi() function)
  • TagsFromTrackNameEvents – use only track names to resolve mapping, useful for midi contained named tracks
  • desiredLength – optionally cut patterns in equal length
Returns:

a list of GSPattern build from Midi folder
