Detailed Description

Class representing a pattern made of GSPatternEvent.

hold a list of GSEvents and provide basic manipulation function

Parameters
durationlength of pattern usually in beat, but time scale is up to the user (can be useful if working on 32th note steps)
eventslist of GSPatternEvent for this pattern bpm:origin BPM for this pattern (default: 120)
timeSignaturelist of integer representing time signature ; i.e [numerator,denominator]
